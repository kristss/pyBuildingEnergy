EN 15316-2:2017 Emission System Implementation Audit

This page maps the new emission-system code to BS EN 15316-2:2017, Energy performance of buildings - Method for calculation of system energy requirements and system efficiencies - Part 2: Space emission systems (heating and cooling). It is intended for tandem review of the standard, code and generated CSV/HTML outputs.

Scope In The Code

The implemented module is src/pybuildingenergy/source/emission_15316_2.py. It covers the hourly calculation path for space-heating emission module M3-5 and water-based space-cooling emission module M4-5. The examples place it between ISO 52016 building needs and the EN 15316-4-2 heat-pump generation calculation.

Example Integration

The examples in examples/heat_pump_15316_4_2_example.py run:

  1. ISO 52016 at the normal heating and cooling setpoints.
  2. ISO 52016 at the EN 15316-2 equivalent setpoints.
  3. EmissionSystemCalculator.run_timeseries() to calculate losses and auxiliaries.
  4. EN 15316-4-2 heat-pump generation using Q_H_em_in_kWh and Q_C_em_in_kWh as the delivered space loads.

EN 15316-2 is the default example mode. To reproduce the earlier direct ISO52016-to-heat-pump workflow, run the example with --emission-method simple.
